	OSI Model (Open System Interconnect) (Network Reference Model)
It Gives overview of entire network system.

It will present you with simplified essential commponents and describes
how these components interact with each other through common network framework.

OSI Model also simplifies problem of n/w technician to easily troubleshoot 
their problem referring it.

	It consist of 7 Layers 
		1. Application
			A. End User Layer 
			B. HTTP,FTP, IRC, SSH, DNS
		2. Presentation
			A. Syntax Layer
			B. SSL, SSH, IMAP, FTP, MPEG, JPEG
		3. Session
			A. Synch & Send to port
			B. API's, Sockets, WinSock
		4. Transport
			A. End-to-end Connections
			B. TCP, UDP
		5. Network
			A. Packets
			B. IP, ICMP, IPSec, IGMP
		6. Data Link	
			A. Frames
			B. Ethernet. PPP, Switch, Bridge
		7. Physical
			A. Physical Structure
			B. Coax, Fiber, Wireless, Hubs, Repeaters
